#  1.Telegram简介
-  Telegram号称是最为 **“自由而安全“** 的通讯软件，
它在通讯中实现了用户匿名、信息端对端加密、聊天信息定时销毁、
“阅后即焚”等功能。
-  实际上，传播色情淫秽信息早已成为匿名社交软件的通病，
而像Telegram这样，用过硬的技术将信息加密做到极致，
很容易成为”懂的都懂“。

#  2.Telegram下载和安装

-  1.安装选择EngLish -> Next  (安装过程是没有中文语音的)
